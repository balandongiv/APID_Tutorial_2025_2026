Session 7 Revision Report – Inheritance and Polymorphism
Chapter Overview

Chapter file: ses7.tex

Associated Python file(s): session7/session7_iris_template.py and any modules created under a session7/ folder.

Intended beginner takeaway:

Recognize why inheritance is useful: avoid duplicating shared code by moving common evaluation logic into a parent class and leaving only prediction rules in child classes.

Implement a simple inheritance hierarchy for Iris classifiers, with a ClassifierBase parent and concrete RuleClassifier and NearestCentroidClassifier children.

Understand polymorphism: different classifier objects can be evaluated in the same loop because they share the same interface.
